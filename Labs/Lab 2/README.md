# Lab 2 - Music Playlist Arrays

## What this project is

A simple JavaScript project that practices array basics using a music playlist.

## What you do

You work with arrays to:

* Store songs, artists, genres, and favorites
* Get first, middle, and last songs
* Add and remove songs
* Loop through songs
* Search for songs
* Use destructuring

## Main array methods used

* `push()` / `unshift()` (add items)
* `pop()` / `shift()` (remove items)
* `forEach()` (loop through items)
* `map()` (create new list)
* `indexOf()` / `includes()` (search)
* destructuring (get values from arrays)

## How to run

1. Open `index.html`
2. Open console (F12)
3. Click buttons or refresh page
4. Check console output
